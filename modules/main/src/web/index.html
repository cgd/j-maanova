J/maanova 1.1.0 - Build 1

This page contains some high-level version and installation information about J/maanova. The information in this page is intended for an internal Jackson Lab audience.

Installation:

Requirements:

  • Windows XP or Mac OS X >= 10.5
  • Java Runtime >= 5.0 (If you are running Mac OS x then Java is already installed. If you need to install Java for Windows go to http://www.java.com)
  • R 2.12.* (you can obtain a copy of R from http://r-project.org)

Click here to launch the J/maanova installer

Version 1.0.0 - Build 1 (SVN Revision: 9143)

This is the first internal release candidate for J/maanova. It includes the following major features.

  • File Input/Output
    • Load Tab Delimited or Affy Data
    • Load/Save Projects
    • Export R Data
    • Export R Script
    • Save Graphs to PNG
    • Save Test Table to CSV
  • Analysis
    • Model Fitting
    • Model Testing
  • Graphs
    • Vocano Plot
    • Array by Array Intensity Scatter Plot
    • Residual Plot
  • Tables and Reports
    • Experiment Design
    • MAANOVA test results table
    • Gene List
